# valid-gtin
A zero cost lightweight validator function in native javascript for GTINs

# Usage
```javascript
const validate = require("valid-gtin")

if (validate(GTIN)) {
  console.log(":)")
}
```

# CLI
`valid_gtin <gtin>`

# Development

To run the tests please checkout this repository and run:

`npm run test`
